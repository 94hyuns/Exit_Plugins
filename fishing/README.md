## 명령어

| 명령 | 설명 | 권한 |
|---|---|---|
| `/낚시` | 낚시 상점 GUI 즉시 오픈 (테스트용) | OP |
| `/계절 [봄\|여름\|가을\|겨울]` | 계절 조회/설정 | 조회는 누구나, 설정은 OP |
| `/낚시리로드` | config.yml 재로드 + 주기 재적용 | OP |

## Shop과 연동

서버 운영자가 작물/광물/램프 상인과 함께 **낚시 상인**을 스폰:

```
/shop spawn fishing [스킨닉네임]
```

플레이어가 낚시 상인 NPC 우클릭 → `FishShopProvider.openShop` 호출 → 계절별 낚시 상점 GUI 오픈.

## 시즌 시스템

- 4계절 (봄·여름·가을·겨울) 순환, 기본 10분 주기
- 제철 6종 (×1.5 판매가), 반대 계절 6종 (×0.5), 중간 12종 (×1.0)
- 현재 계절은 `plugins/Fishing/state.yml` 영속화
